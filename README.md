# Leveraged Buyout Analysis – Redington Limited

## Overview

This project evaluates the acquisition of Redington Limited through a Leveraged Buyout (LBO) transaction. The analysis focuses on transaction structuring, debt financing, operating performance forecasting, leverage optimization, and sponsor return generation over a five-year investment horizon.

## Transaction Summary

| Metric | Value |
|----------|----------|
| Enterprise Value | INR 9,096 Cr |
| Sponsor Equity | INR 3,711 Cr |
| Debt Financing | INR 5,567 Cr |
| Exit Equity Value | INR 15,990 Cr |
| Money-on-Money Return | 4.3x |
| IRR | 34% |
| Holding Period | 5 Years |

## Model Components

- Historical Financial Analysis
- Assumptions & Drivers
- Entry Valuation
- Sources & Uses
- Operating Model
- Debt Schedule
- Return Analysis
- Sensitivity Analysis

## Investment Thesis

- Market-leading technology distribution platform across India, Middle East, and Africa.
- Asset-light operating model with strong cash generation.
- Stable EBITDA expansion supporting deleveraging.
- Attractive sponsor returns under base-case assumptions.

## Key Takeaways

- Generated a projected 4.3x MoM Return and 34% IRR.
- Reduced leverage through scheduled debt repayment.
- Created value through operating growth and multiple expansion.
- Demonstrated end-to-end LBO modelling and transaction analysis.
